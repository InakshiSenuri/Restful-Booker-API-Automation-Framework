# Restful Booker API Automation Framework

API testing framework built using:
- Java
- Rest Assured
- TestNG
- Maven

Includes:
- CRUD API automation
- Postman collection
- POJO serialization/deserialization
- Dynamic booking creation
- Authentication handling

## Tech Stack

- Java 17
- Maven
- Rest Assured
- TestNG
- Postman

## Covered Scenarios

- Create Booking
- Get Booking
- Update Booking
- Partial Update Booking
- Delete Booking
- Authentication
- Negative Scenarios

## Run Tests

mvn clean test
